📦 Painel Revenda SSH (Versiónes)

Painel Revenda SSH (Versiónes) es un repositorio centralizado que reúne múltiples versiones del Panel Revenda SSH para sistemas Linux. Este conjunto de scripts permite instalar, actualizar o probar distintas ediciones del panel, adaptándose a distintas necesidades de usuarios y entornos VPS.

Está orientado a revendedores y administradores que desean elegir la versión que mejor se adapte a su flujo de trabajo, ya sea por compatibilidad, estabilidad o funcionalidades específicas.

✔️ Características

- Instalación automatizada de diferentes versiones del Panel Revenda SSH (v10 a v25).
- Opciones para actualizar versiones antiguas a más recientes (por ejemplo, de v23 a v25).
- Herramientas adicionales para limpieza de archivos, desinstalación y mantenimiento.
- Menú interactivo de selección simple y directa.

💡 Ideal para quienes desean probar versiones anteriores o mantener sistemas con configuraciones específicas.
